# City 311 Analytics Project

## Overview
End-to-end analytics project simulating a city’s Innovation Division analyzing 311 service requests to improve operations, response times, and citizen experience.

## Tech Stack
- BigQuery (data warehouse)
- dbt (data transformation & testing)
- Qlik (primary dashboards)
- Power BI (supplementary visuals)
- Python (synthetic data generation)

## Data
Synthetic datasets (25K+ records) including:
- Service requests
- Request updates
- Citizen feedback
- Categories & neighbourhoods

## Key Features
- Data pipeline: raw → clean → reporting layers
- KPI development (response time, backlog, reopen rate, demand trends)
- Interactive dashboards for operational insights
- Predictive modeling for request volume trends

## Outcome
Provides actionable insights to optimize city operations, resource allocation, and service performance.
